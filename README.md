# Estudo de Arrays em Python 

Este projeto é um exercicio pratico focado no modulo nativo `array` do python, explorando a manipulação eficiente de dados numéricos.

## O que foi aprendido

Neste script, explorei os seguintes conceitos e métodos:
- **Criação de Arrays**: Diferença entre listas e arrays tipados (tipo 'd' para double).
- **Manipulação**: Métodos `append()`, `insert()`, `pop()` e `remove()`.
- **Busca e análise**: Métodos `index()`, `count()` e fatiamento (slices).
- **Persistencia de dados**: Gravação e leitura de arquivos binários com `tofile()` e `fromfile()`.

## Como executar
Certifique-se de ter o Python 3 instalado. no terminal, execute:

```
python3 exercicio_arrays.py
```
